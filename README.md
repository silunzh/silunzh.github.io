# Silun Zhang Academic Homepage

This is a static GitHub Pages version of the previous Google Sites homepage.

The design direction is:

- Academic Pages-style structure: profile sidebar, top navigation, dedicated academic pages.
- Murray-style compact homepage: quick links, direct sections, dense academic information.
- No build step: GitHub Pages can serve these files directly.

## Pages

- `index.html` - compact homepage
- `research.html` - research overview
- `publications.html` - publications
- `group.html` - group members and open positions
- `courses.html` - courses and mentoring
- `services.html` - academic services
- `contact.html` - contact information

## Publish on GitHub Pages

Create a repository named `USERNAME.github.io`, where `USERNAME` is your GitHub username, and upload the contents of this folder to the repository root. GitHub Pages will serve `index.html` as the homepage.

If publishing from another repository name, enable GitHub Pages in repository settings and choose the branch/folder that contains these files.
